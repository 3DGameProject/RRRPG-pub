# 기능 세부

### 플레이어 이동
플레이어가 직접 조작하지 않아도 플레이어가 앞으로 나아갑니다.

### 기본 UI 구현
플레이어가 오브젝트에 부딪히면 UI에 반영됩니다

### 개임 내 통화 시스템
화면을 클릭해서 재화(개미)를 모을 수 있습니다.
